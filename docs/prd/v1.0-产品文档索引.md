# 极简栈创造 v1.0 产品文档索引

## 项目背景

**极简栈创造** 是一个面向内容创作者的一站式 AI 辅助短剧生产平台。用户从一句创意描述出发，经由 AI 自动生成剧情大纲、角色圣经、场景圣经、分镜脚本，逐集产出完整的短剧视频。

## 文档拆分说明

原始产品文档按功能模块拆分为以下独立 PRD，每个 PRD 聚焦一个可交付模块，包含执行摘要、用户故事、功能说明、验收标准、排除项、依赖关系和风险分析。

## 全局约束

1. 所有表预留 `user_id` 和 `team_id` 字段，默认填充系统单用户 ID。
2. API 预留 `Authorization` Header，当前直接透传不做校验。
3. 每个 AI 生成节点右上角提供「模型选择器」下拉框，可独立切换模型。
4. 所有可编辑节点提供「版本历史」功能，支持 diff 对比和回滚。
5. 用户修改字段后，系统实时提示变更影响范围。
6. AI 不自动推进到下一步，必须用户手动触发。

## 优先级规则

- 优先级必须全序排列，不允许并列 P0。
- 当前 Sprint 内 P0 需求总数不超过 3 个。
- 资源冲突时主动召集受影响角色评审。

## 验收标准通用要求

- 所有验收标准必须可量化、可测试。
- 性能指标：文本 AI ≤ 30 秒、图像生成 ≤ 60 秒、视频生成 5-8 分钟/节点、合成 ≤ 5 分钟/集。
- 可靠性：生成失败自动重试 3 次，断点续传，页面刷新不丢失进度。
