# YZ-20 前端功能 E2E 测试计划

## 测试范围

基于 PRD `v1.0-ai-chat-web.md` 和 OpenAPI 文档 `openapi.yaml`，覆盖 YZ-20 新增的全部前端功能：

1. **AI 对话流式输出** — 多轮文本对话 SSE 流式接收、打字机效果、消息类型切换
2. **图片/视频生成** — 图片生成同步返回、视频生成异步提交与进度轮询
3. **用户中心** — 查看/编辑个人信息、头像上传（预签名 URL → OSS 直传）
4. **订阅与额度** — 计划展示与对比、当前订阅与额度用量、取消订阅、账单历史
5. **会话管理** — 列表增删改查、搜索过滤、重命名、删除
6. **文件上传** — 图片附件上传流程

## 测试类型

- E2E 测试（Playwright MCP）
- API 接口测试（curl + 响应结构验证）
- 数据完整性验证（MySQL MCP，staging 环境）
